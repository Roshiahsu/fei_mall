# 專案名稱fei_mall
此專案是簡易的商品購物平台後台，使用SpringMVC進行專案設計

## 專案介紹
此專案簡易的電商系統，包含前台商城系統及後台管理系統，基於SpringBoot與MyBatis實現。
- 前台商城系統
  - 首頁
  - 商品推薦(熱門商品、優惠商品、新上架商品)
  - 商品搜尋
  - 商品詳情
  - 購物車
  - 訂單流程
  - 會員中心
- 後台管理系統
  - 訂單列表
  - 商品管理
  - 品牌管理
